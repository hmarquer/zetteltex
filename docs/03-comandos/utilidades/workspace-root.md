# Opcion global --workspace-root

## Proposito
Definir la raiz del workspace sobre la que opera cada comando.

## Sintaxis
`zetteltex --workspace-root <ruta> <comando>`

## Requisito estructural minimo
- notes/slipbox
- projects
- template

## Ejemplo
```bash
zetteltex --workspace-root /ruta/a/mi-workspace list_projects
```
